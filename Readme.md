# DhyanSport

DhyanSport is a static website for a sports and research organisation, featuring information about programs, events, membership, research, internships, and contact pages.

Live demo: https://dhyansport.netlify.app/

**Features**
- Clean, mobile-friendly HTML/CSS pages
- Sections for Home, Programs, Research, Events, Membership, Finance, Governance, and more
- Static assets (images) included in the `Assets/image` folder

**Live Link**
- https://dhyansport.netlify.app/
